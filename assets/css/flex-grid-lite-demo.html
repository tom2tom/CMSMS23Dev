Flex Grid Lite

The Functional Flexbox Column Grid

Flex Grid Lite is a lightweight column grid with all the power of flexbox. Built using a functional CSS approach which means you get exactly what you ask for without any hidden surprises. The resulting file size is far smaller than a traditional column grid.

Basic Layout

Each cell inside a grid row is by default assigned an equal share of the total width. The cells are configured to respect the layout over the content.

Gutters

Padding is built into the grid. The default size is 8 pixels (vertical and horizontal). The amount(s) can be modified by setting margins and padding of relevant css classes. Padding can be removed by adding class .nogutter to the relevant row(s).

Fixed Widths

Add in the fixed width class and pass the cell a defined width or allow content like an image to dictate the desired width. It's recommended to fill the space with dynamic flex cells however you can mix in percent-based cells too. Utilizing a fixed column with only percent-based cells will make it difficult to reach 100%.

Media Breakpoints

Media breakpoint classes define the screen size below which cells go to 100% width. The small .cell_sm snaps to 100% at 480px, .cell_md at 624px, and .cell_lg at 744px. The breakpoint classes will override any other percentage- or grow-based modifier class.
