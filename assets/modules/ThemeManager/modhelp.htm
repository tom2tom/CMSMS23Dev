What Does This Do?

This module helps you to export, import and manage theme packages, which can include:

  • smarty templates
  • stylesheets
  • stylesheet groups
  • tags
    • regular CMSMS smarty-plugins
    • user-plugins (a.k.a. user-defined tags)
  • media files (images, movies, sounds)
  • custom fonts
  • scripts

How Do I Use It

This module has no front end interface, only an admin interface. It allows selecting any of the above-mentioned components, enter package info, and click "Export". An XML file containing all the components of the package will be created for download. That file can be distributed to import the package into other CMSMS installations.

Permissions

A strict permissions model is enforced for the Themes Manager, to ensure database integrity. The permission "Manage Themes" is required to export themes and import themes.

As well, the opposite functionality exists, you may upload a package file (xml format) and have the enclosed templates, stylesheets, pages, module templates, global content blocks, user defined tags, module settings and files automatically imported into your CMSMS installation.
